	Идея 1: Сделать массив, состоящий из 1 для уменьшающихся радиусов и -1 для 		увеличивающихся и устроить поиск по нему;
	Идея 2: Сделать кортежи: (значение, индекс) и сортировать по значению.
РЕШЕНИЕ: По первой идее написано условие на нахождения -1 в отрезке массива
	 (от предыдущего максимума) так, чтобы последний элемент наоборот не был -1.
